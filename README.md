# Moshe Uziel - Personal Branding Website

A minimal, content-first personal authority website for Moshe Uziel, focusing on AI strategy, governance, infrastructure, and agentic systems for complex organizations.

## Overview

This website serves as an authority hub for thinking around AI strategy, infrastructure, governance, agentic systems, healthcare AI, enterprise AI, and public-sector AI transformation. It's designed to be fast, SEO-optimized, and content-focused rather than a CV site or generic consultant landing page.

## Features

- **Fast Performance**: Built with Next.js 15+ and static rendering
- **SEO Optimized**: Comprehensive meta tags, structured data, and fast loading
- **Responsive Design**: Mobile-first design with Tailwind CSS
- **Content-First**: Focus on essays and analysis rather than personal promotion
- **Professional Aesthetic**: Clean, minimal design that emphasizes authority and expertise

## Tech Stack

- **Framework**: Next.js 15+ with App Router
- **Styling**: Tailwind CSS
- **TypeScript**: Full TypeScript support
- **Icons**: Lucide React
- **Deployment**: Designed for Vercel

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Deployment

The site is configured for deployment on Vercel at https://vercel.com/moshe-uziel-s-projects.

### Environment Setup

1. Connect GitHub repository to Vercel
2. Configure custom domain: mosheuziel.com
3. Set up analytics and monitoring
